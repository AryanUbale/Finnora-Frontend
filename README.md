# Finora Frontend

Frontend for Finora — an AI-powered fintech SaaS platform built using React, TypeScript, Vite, and Tailwind CSS.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Zustand
- Recharts

## Features

- Authentication System
- AI Financial Dashboard
- Budget Management
- Transaction Tracking
- Analytics & Charts
- OCR Receipt Scanning
- Dark/Light Mode
- Responsive SaaS UI

## Run Locally

```bash
npm install
npm run dev
